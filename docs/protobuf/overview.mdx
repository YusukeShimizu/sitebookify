## Source of truth

All Protobuf schemas live in `proto/`.

- Buf module configuration: `buf.yaml`
- Dependency lockfile: `buf.lock` (generated via `buf dep update`)

## Dependencies

This repository uses:

- Validation annotations: `bufbuild/protovalidate`

## On-disk schemas

Sitebookify stores most data as JSONL / YAML / Markdown.
The Protobuf files in `proto/` are used as schema references for on-disk formats (not as a public API).

- Manifest schema: `proto/sitebookify/v1/manifest.proto`
- TOC schema: `proto/sitebookify/v1/toc.proto`
